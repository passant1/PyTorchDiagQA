# 实验三：文本分类

本文件夹包含实验三所需的代码、数据和运行结果。

- `src/`：TextCNN 文本分类代码
- `data/text_classification_corpus.csv`：从 `199801` 大语料按关键词抽取的体育、科技、教育弱标注数据，共 180 条
- `outputs/exp3_classification_result.txt`：已生成的分类结果
- `outputs/exp3_loss_curve.png`：训练损失曲线
- `outputs/exp3_confusion_matrix.png`：混淆矩阵图

运行命令：

```bash
python run_exp3.py
```
